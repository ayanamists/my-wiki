谢谢友友们，大家对我的孩子的姓太关注了，我招架不来了。我决定以后我没有姓了，谢谢大家，我的孩子以后也不会有姓的，这样就没有什么冠姓权之争了。我会想个办法去掉我的姓，争取户口本上没有姓氏只有名字，这样希望大家满意，谢谢，我成功了的话我会开个帖子分享我的经历，有希望一起去掉姓氏的友友可以联系我，我们一起想办法。希望大家不要再说我放弃自己的权利了，没有姓就没有纷争了。我爱这个世界，希望大家可以和平。

有的人跑来我这里骂了我一顿然后把我拉黑了......无语住了，说男权女权就离开吧，我只是想问一下我怎么做比较好，不讨论男权女权这些高大上的东西

再补充一点：为什么不和父母断绝关系？

回答：断过了，但是结婚我总不可能绕过父母结吧，至少户口本在他们那里啊。为什么这段话是妈妈发的给我断绝关系的截图？因为我前面发疯指责她后，删除了对话框，她后来给我中间一段涉及到我爷爷奶奶，不想老人家牵扯到。发的这些话。怕被熟悉的人看到，码打得厚。我确实很难说断绝关系的话，但是明确说明了少来往。有半年多我们一句话都没说过，后来是我妈妈住院我才打过一次电话，直到最近有结婚打算。

![image](https://img2.doubanio.com/view/group_topic/l/public/p623578361.webp)

不止是这种时候，我以前也明确说过不愿意他们参与我的生活的话，我要自己做决定，他们提出不合理的要求，我没有一次同意过。包括在我刚工作工资只能勉强自己够用的时候要每个月1.5k的赡养费，省钱给他们用，要求我帮忙还贷款，要求我去某地医院任职等等。我没有同意过，也没有给过他们钱。不知道你们怎么界定断绝关系？一定要有一个宣言吗？那我确实没有这个宣言。我只是尽可能远离他们，不和他们有来往，他们对我做的很多事情，我也发疯发过了，但是他们要么不以为然，要么说：没想到你是这么想的。从我大学后，我们每年只有过年见一面，平时电话更是几乎没有，我没去过我外婆家，我也不去他们所在的城市，我所说的没有夸大事实，这是真实的发生在我身上的事情。

补充一点，楼主家里两个女儿。顺便问问大家，我男朋友来我爸妈工作城市，我爸妈完全不管要求我男友安排行程吃饭这是妥当的吗？？

楼主ip地懂得都懂，很看重对象是不是本地人，不巧，男友是很北的北方人。不过目前已经考上了本市国企，想着工作也稳定了可以带去见父母了。父母在隔壁市，到时候我和男友一起高铁到隔壁市。

结果，我问了一个阿姨，她说需要我男友全程安排去隔壁市吃什么玩什么，然后带礼品过去，我一头雾水，按道理来者是客吧，我男朋友作为第一次上门的人，居然还替主人家安排行程啦？总感觉怪怪的，但也没多说什么，打算就这样去。下午和妹妹聊天，问爸妈那边怎么想的，结果妹妹告诉我，爸妈一直在谈论彩礼要多少钱，说身边谁谁谁68w彩礼好有面子，说某某虽然没结婚但是这个条件在我们村可以拿百八十万彩礼。其实很早以前我和男友出去旅游，那时候我妈妈就提过彩礼要多少w，我问那我们要多少嫁妆？结果说我们村就是一分嫁妆都不会出的，说真的我听完有一种抬不起头的感觉。这次听到，心里更是心烦意乱又生气，面也没见，就先谈上彩礼怎样了，好像我是个货品，养到这么大，该卖了。

和之前那个阿姨聊，我和父母关系一般，我就想跟她谈谈我的看法，希望她能代我传达给我妈，大意就是这个彩礼我不想要，ip地房价高，我自己打拼买房子，想把彩礼用作买房的钱，而且我觉得就是在卖女儿。结果刚说完，阿姨就气冲冲地说，那你父母养育之恩不报？你只想到自己，你自私，你不管父母，别跟我说什么歪理。遂拉黑。

彻底崩溃，哭了一小时。

写一下最后关于养育之恩的事。我九岁前一直由爷爷奶奶带大，我父母除了逢年过节带我玩一下买点东西打打电话完美隐身，说我爷爷把我抱走不给他们养，但我对此存疑。小时候也就算了，年纪大一点真的不能带我走？九岁那年突然就把我带走了，送到住宿学校，我吃不惯学校的饭，饿了三年。我曾经跟父母说过在学校吃不饱，但没有下文，会买零食给我，长大了问为什么不走读，理由是我发育早，饿一饿想着可以控制发育......初中送回老家，也是放在亲戚家，亲戚做家教的，甚至曾发生过让我在上学报到前一天去乡下爷爷家要学费的事情。高中因为要住宿，所以可以回爷爷奶奶家，这期间，学费父母承担，生活费由爷爷奶奶承担。大学是他们承担，在上海实习，医学生所以没有实习工资，我妈妈给我的钱去掉水电费房租通勤之后1000不到，又过上了吃不饱饭的日子，难以想象吧，2021年，在上海实习的小姑娘，吃不饱饭。还好男朋友会给我一些钱吃饭。工作第一年，没有钱租房，我想向他们借3000应急，也拒绝了。我曾想过向有钱的同学借，但是也很难借到，毕竟人家也要生活。最后这笔钱是我男朋友支付的。爷爷奶奶知道了，提前预支了工资，给了我2600，说先给我应急。

说这么多，不是卖惨，只是说明我长到现在，父母给过帮助，但绝对不多。不仅仅是钱，也无视我的健康。我高三生病，一年多没来姨妈，跟妈妈说，也没说带我看病，最后还是婶婶知道了，说要带我去，我妈妈才说要带我看病的。

乱七八糟的，谢谢各位看到这里。只是我不明白，我不想要彩礼，想要买房（不需要我父母帮助，自己和男朋友打拼）难道错了吗？这就算是自私了吗？
