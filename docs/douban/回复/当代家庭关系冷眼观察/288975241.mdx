标题党了，也是事实。

------------

再更新一下，未免误会，我的身份是自私的姐姐。如果有兴趣吃瓜，最后附上曾经极度崩溃的时候吐槽的帖子。看过就了，不必在意，现在过得很好。都过去了，别上头了。

深更半夜母亲打电话向我抱怨此事，说妹妹二阳了回家隔离，第二天奶奶就去了亲戚家住，让妹妹叫同样七十几的外婆来照顾。母亲很生气，话里话外觉得奶奶的行为太过分，奶奶自己出去二阳了还责怪是妹妹的锅，妹妹对此也有点怨气。

而之前我也接到奶奶的电话，说大伯听说我妹妹阳了，强行把奶奶留在他家。我劝她离我妹远点，别被传染了。妹妹已经成年，家所在的地方外卖、医疗、交通都非常便捷，她完全能照顾好自己。

很疑惑，他们好像都想在我这里寻求某种认同。我奶奶阳的时候，妹妹也是躲在自己屋里，一报还一报？

不知道什么样的回答他们才会满意？

------------------

更新一下，

自私和虚伪的血液流淌在我们家每一个人的血管里，对于这事儿我也像个看客，但是充满了疑惑。在以往，里面会多一个角色，我去照顾妹妹，我带奶奶去医院，我给做完手术的爸妈陪床做饭。最后我受不了了，跑路了。看他们互相埋怨，指责，和我说了又有什么用呢？

奶奶极度重男轻女，母亲年轻时受了非常多的伤害，甚至前几年差点因为奶奶婚姻破裂。她为啥会指望一个讨厌多年的人会善待自己的女儿？

奶奶我就更不懂了，来我面前解释，我难道会为了一个从小就不喜欢我的人去驳斥我妈吗？确实提了两句，然后我妈三天没理我。

好像身体逃离了，精神还在过去。
