这里的帖子好像大多是和父母姐妹有矛盾的，我来叙述下刚才和儿子发生的冲突吧。

儿子十四岁，这个暑假是八升九的关键期，但他自制力差，到现在老师布置的作业已经落下了很多

因为他数学和英语都不好，如果不补的话可能考不上目标高中，所以暑假初我就和他商量，一定要报个数学班，他也同意，当时给了他几个选择，经过商量我们最终选定了一个，明天开始上课，全天班，十天，内容主要是预习。

然后就在刚才，他在磨蹭着吃了一个小时饭后开始哔哔，主要意思为，十天太久了，他会死的，预习没用，反正学校里都会教，他觉得没有意义，等等等等，车咕噜话说了很久，一开始我还忍着，后来烦了，明确告诉他他数学成绩不好，一定要补，这个班是暑假开始就商量好报的，他不满意为什么当时不说，要报这个，请他闭嘴，不要再说了。但他一个劲继续说，各种歪理说预习没有意义，这个班不好，在我多次让他闭嘴后还在说，我真的很烦了，摔了东西，用衣架抽了他，赶他去阳台待着了。

我知道动手不对，可实在控制不住情绪了。
