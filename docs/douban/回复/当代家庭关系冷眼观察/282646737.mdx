从小我爸就对我很严，没上幼儿园的时候，让我背九九乘法表，没有背出来就打。如果说这算很正常的家庭教育，但是他们还常常告诉我"男子汉不要婆婆妈妈的"、"大男人哭什么哭"、"你xx怎么跟个女人一样"。这几句，或者说类似的话，从小就让我知道，男性对女性的一些厌恶和轻视是根深蒂固的。但这也让当时我这个小男孩很苦恼，我不开心了真的想哭，也真的会有很多话要讲------这也就是为什么当代男性很少表达自己的情感，被规训得很厉害。

时间转到我六年级的时候，我爸因为一些事情，像变了一个人------也有可能是因为我太久没见他了，他在我小时候就这样。他每天都会找许多奇奇怪怪的理由殴打我，体罚我。等到4点左右，我母亲快下班了，他就假装原谅我，然后哄我。我母亲到家看到我不高兴，他就会替我否认，并且让我笑一笑。我必须照做，我试过告诉母亲实情，但第二天他会告诉我后果是什么。我的家就像网瘾学校，而他就是那个杨永信。有时候他会自己上厕所不冲马桶，然后说是我上厕所没冲，以此为理由殴打我。我跟同学抱怨学校伙食差，他会直接拿大西瓜砸我的头，想了半天想出一个理由："你同学到时候告诉校长，你在学校就完蛋了，我现在是给你个教训。"还有更多的称不上理由的理由，比如桌上有张餐巾纸等等，希望你不会觉得我在编故事吧，如果这里的人都这么想，那我真的没有可以倾诉的地方了。

过年时，我经常和我爸妈去外婆家。那时候每次一见到外婆，外婆总要抱住我的头一顿亲，我觉得很反感很讨厌，每次想挣开都挣不脱，那时候已经十岁有余了。后来有一次，家里没什么人，外婆把自己的衣服撩起来，给我展示她的乳房。当时还没有开窍，以为外婆只是想开个玩笑，而我是那个在小学被校园暴力时被定为"开不起玩笑的傻子"。所以当时我觉得是我自己的问题，是我开不起玩笑，眼睛像看到正午的太阳一样，捂着脸逃跑了。后来我听说了许多女孩子在原生家庭的遭遇，我才反应过来，这多年来如芒刺背的东西，不是我开不起玩笑，这就是性骚扰。现在回首，那些事让我感到恶心、害怕，我甚至有时想死在他们面前，告诉他们是你们害死我的。可怕的不止这件事本身，当我发现没有人相信男孩会被性骚扰这件事，这种恐惧它加倍了。

从初中起，我发现我喜欢女孩子的衣服，因为我讨厌男装千篇一律的设计和黑白灰。直到上了高中，我自己赚了一点钱，也有了自己的银行卡，于是买了一些裙子或者女性向的东西藏在家里。同一时间，学校里有一个gay在对我进行性骚扰（必须要提一下，我意识到他在对我性骚扰这件事，是经过我一个女性好友提醒，我才醒悟的）。一天晚上，他说了一些很不堪入目的话，我就回了三个句号就不回了。过了没几天，放学回家，我爸突然让我罚站，站到我承认错误为止。我当时哪知道他连续几年偷看我的手机（上学时我手机放在家里，我设置自己的虹膜解锁，他能用他的眼睛解开，导致我至今能不开面部识别，就不开）。我那时候膝盖有很严重的伤，站五分钟都站不住，每一秒都是剧痛，而他让我站了两个小时以后，一脚把我踢倒在地上。他说他看了我的手机，问我为什么面对别人gay的骚扰，不骂他，不拉黑。我说我发三个句号就代表我反感他的话，没有要接受他。他开始骂我恶心、yd，他还说"你是不是很喜欢被男的用x插py"。好恶心，生理性的恶心，我每次想到这个事我都想死。

高三临近高考（2020），他看到我的学习成绩稳步向前，随便找了一个理由，把我上网课，做作业用的笔记本砸了（当时手机完全不能在网上做作业），用拳头疯狂重击我的头。当时他打得很厉害，我甚至以为我要死了，因为他另一只手掐着我的脖子。我当时也释然了，我不再受"父母不管对你如何，你都要对他们好"的愚孝思想的限制，想和他同归于尽。还是我母亲和奶奶把我们拉开了。三年过去了，他一直向我道歉，我觉得那是更大的阴谋。

我妈也没有好到哪里去，她让我觉得我只是她一个喜爱的宠物。我从小就不喜欢吃虾，但是她又经常买了烧给我吃，并说这都是给你烧的（虾的营养有一堆其他我喜欢的食物可以代替）。有一次，她会问我一遍吃不吃，我说我不吃。连续问了我三遍之后，我都说不吃，她便按着我的头，用筷子夹起一只虾往我嘴巴里硬捅------前一秒她像爱孩子的妈妈，后一秒她就成了面目狰狞的魔鬼。这件事只是她与我之间无数事件的代表性案例之一。

近几个月，我父母卑微着向我道歉，求我回家过年，但是我长大了，不再是那个给两颗糖就会心软的小朋友。
