一个比较残酷的事实是，目前想进互联网公司的话，还是比较看学历的。同样地，如果你在比较好的学校的话，那应该还是不用太担心就业的问题。（当然，这不是说你什么都不会就能进去，还是要好好准备面试的）
至于你说「文科在我看来是很高大上的，都是知识分子，而我们理科生都是打工的」，我觉得还是有必要澄清一下。（理论）计算机科学的源流是数理逻辑中的递归论（recursion
theory）.
逻辑学既是数学的一个分支，又与哲学有很大的联系。与计算机科学关系比较大的逻辑学家，如
Church、Kleene、Gödel
等等，在哲学上也有一些建树。理科生不是「打工的」，起码就我们计算机科学来说，它同样可以教给你一些「真正的智慧」。
至于学 java
或者什么的，我觉得这种泛泛而学最好是避免，起码我的经验是会非常快的厌烦，然后放弃。最好的学习方法还是你想要做点什么东西，比如说做一个网站、做一个服务，然后你去学需要的技术。现在
ai 也很智能，你可以多问问 ai. 如果想「补一补课 」，那不如学学 sicp\[1\]
之类的，感受一下计算机科学的优雅和美丽。 \[1\]:
