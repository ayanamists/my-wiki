1.  在经典逻辑里， p / q -\> \~p -\> q 是永真式，可以用真值表检查
2.  在直觉主义逻辑里，p / q 可以解释成 "要么能证明 p，要么能证明 q"，\~p
    的解释是 "不能证明 p"，把这两个条件拼起来，得到 "一定能证明 q"
