钟离、温迪自不必说，传说任务到现在仍然是最高质量

魈，用了璃月主线炸群玉阁（魈抱了旅行者）------传说任务（并肩战斗）------海灯节（旅行者把海灯节搬到了望舒客栈）来一步一步塑造魈和旅行者之间的羁绊，最后逐月节魈随旅行者的呼唤而来，那一句"什么事？"，就让克制的暧昧氛围达到极点。不需要赤裸直球的表白，通过委婉的台词暗示微妙的情愫，这种东方文化的含蓄表达------就，很妙(/ω＼)。

万叶在1.6出场，到稻妻在主线有极具震撼力的剧情高光，整个剧情线特别完整。

阿贝多的雪山剧情也特别好，画面，分镜，情节，几乎都可以说是完美。

目前基本上没有男角色的剧情出现引起巨大争议的硬伤，而且男角色很多样性，各种性格都有，塑造得很细腻。可以说原神的男角色是少而精。

相比之下，女角色这边，传说任务质量高的好像没几个。。。塑造明显敷衍很多，雷神、心海的传说任务大家都觉得不太行。

我猜也许是大部分男玩家只需要单纯的感官刺激就行了，女角色只要胸大腰细腿长，就有一堆人喊老婆，屡试不爽，所以也不怎么需要很认真地打磨剧情，男玩家又占比很高，所以才一个接着一个地出女角色，也来不及好好打磨。既不需要，也没时间，所以很多女角色的剧情才这么粗糙。

以上纯属猜测，不一定对。欢迎探讨。
