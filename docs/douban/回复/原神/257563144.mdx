这破活动所有人的难度一样，大佬直接碾了根本不用看配药，萌新低保都拿不到...卡dps
check又不能慢慢磨，又不能吃药又不能联机。本来难度就高还锁船，给的试用角色也扣扣索索，你给试用肯定是想骗氪的对不对，第一次给个万叶还行，今天这个鹤师傅云师傅刻师傅有人用完觉得很强很想抽吗？我也不赞同乳角色但这用下来只觉得角色强度真的有差...会用试用角色的肯定是萌新，然后这几个试用角色全0命4星武器认真的？

你游本来原石就抠门，活动原石一般默认和日常一样就是低保，结果现在连低保都拿不到...我确实没手但你游又不是什么正经ACT有必要这么为难人吗......打了一下午4银都没拿不到，现在体力全满都不想清看到游戏都烦

-------

跟着B站一个号称萌新攻略的去凹了下发现更不行，第二关都过不去😅

总结下这个活动的屑点，不能吃药加限时导致对DPS要求很高，但绝大多数新人只有一个C练度还不一定够，今天活动给的C本身也不是特别强。而萌新攻略很多虽然用的是新手角色但练度也是真萌新赶不上的...我刚B站抄的攻略要用70级凯亚，笑死，根本没有，现场拉材料都不够....
