我平日里喜好游览山水，行侠仗义，顺便收集古籍孤本。我的收藏中除了志怪传奇，武林宝典，也有不少诗文词曲。有些诗词读起来颇有侠义之气，我很是喜欢，便也背了不少，想着日后写进小说里，或是闲暇之时念给旅行者和重云听。只是不知为何，每每与旅行者同行时，便只能背出"有时明月无人夜，独向昭潭制恶龙"。这诗颇具豪情，我当然喜欢，只是我此前从未见过这句诗，也不知晓作者姓名，这未免有些奇怪。旅行者却告诉我这是来自她的世界的一名诗人所写，如此江湖气概，令我不禁心生向往，还请各位旅行者告诉我更多。
