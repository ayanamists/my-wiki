首先我是女生，男主好色的性格我甚至也讨厌，但谁在乎呢？我看得是剧情的发展和它有趣的世界观。

有一说一，这个番并不适合放在平台上让大多数人接受它，最主要的还是因为它有成人元素在，还有脱离现实世界的正确三观。但万没有想到它能上热搜被骂，而且连同在追番的人也在骂（多半评论还会有人来骂我）。

不是吧？现在这个世纪，是一个心理健全的人，有谁会看什么剧就跟剧里的人三观同理？如果还有人这么认为，那我也不想让你完全认同我，这个社会本就能容纳各式各样的看法。

如果是像我一样在看番的人，请你也不要听信别人怎么去评价你的人格，我们就是随便看看剧罢了，哪有什么坏心思呢
