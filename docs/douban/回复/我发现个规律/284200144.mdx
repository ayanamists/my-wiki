清理桌面发现很多灰尘，仔细一看居然全是断裂的纤维，基本上不是我们小时候的那种灰尘---自然尘土，而是我们的衣服、床上用品等等上面掉落下来的纤维段，很多非常细小，不易察觉，除掉一部分动植物纤维，剩下的不都是现代工业塑料的产物吗？这些东西漂浮在空气中被我们呼吸进身体里面，也就意味着一些化学元素最终会沉淀在我们体内，那些从石油中提炼出来的东西，却回到了我们体内......

再想到那些塑料垃圾，虽然很难降解，但是它们最终也会变成细小的纤维，进入土壤，进入海洋，进入我们的食物中，那未来人类的身体是不是就会越来越差，变成工业元素的最终宿主......
