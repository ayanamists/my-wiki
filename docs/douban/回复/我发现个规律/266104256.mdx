举个栗子:

A愿意为了B做任何事情、任劳任怨不求回报。

大家是不是会觉得A伟大无私崇高？

但是呢，A做这些事是因为ta想这样做，是为了满足自己想为B做事的需求，这一切的出发点是围绕着A的想法，是不是可以理解是为了满足A的奉献欲？

栗子2:

你走在路上发现有人用木棍敲打小狗的头，狗头被打得鲜血淋漓，你看不下去制止了那个人。

你制止ta的原因是你看不下去了还是狗子受不了了呢？

提起利益，大家可能首先想到的是钞票房车这些物质利益，但情感本质上也属于利益范畴，即精神利益。

我觉得所谓的爱、感情之类的被捧得太高了🤔

亲情是为了满足传宗接代、找个情感寄托、免得被人说闲话、从众随波逐流、喜欢孩子、养老、制造一个生活动力等需求；

爱情是为了满足有人陪伴、有人照顾、喜欢美色、传宗接代、对方提供的情绪价值、想和对方在一起、繁衍、⭐欲等需求；

友情是为了满足被理解、被陪伴、被欣赏等需求。
