甚至都没啥特别的"思维"，非说有那也是应试思维

而且学生中也有爱出风头的 沉默踏实的 爱跟老师套近乎的 高冷的活泼的
真诚的圆滑的等等各种人，有的人学生时代已经创业，很多社会人也不一定强于优秀学生的水平和认知层次，硕士博士生留学生年龄无限制
用"学生"概括是否合适？

但是现在全网都在说什么"典型的学生思维""改掉学生思维"让我很迷惑
这些为什么跟学生绑定在一起了

这个词的意思是，学生有一个确定的人生目标（考上好大学），只要好好学习就行。但是如果你上了大学，那这种思维就不太合适了，因为人生没有这样的单向路径。现在这些用法我觉得都是望文生义，和原始含义没啥关系了。
