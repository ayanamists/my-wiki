由于互联网上的"女权话语"\[此处的"女权话语"指的是豆瓣微博上宣传女权主义的话语\](\^1)被女性所掌控，女权主义对于男性的观察必然沦为一种外部观察。"男性基本盘"这个令人啼笑皆非的词语正是这种外部观察的产物。我无意评论这个词反映现象的准确性。但我个人的立场是，男人与男人的区别是很大的，用男人这个身份符号来划定一种形象，套用到一个具体的男人身上，恐怕会相当失真。

不过这个题目本身还是有点回答价值的：作为一个男性，我不期望能找到什么"好的"女朋友。"好的"这个词代表了一种（世俗意义上的）优秀标准，比如学历、家境等等。这是我所不太看重的，咱只关心自己喜不喜欢，不太关心别的。
