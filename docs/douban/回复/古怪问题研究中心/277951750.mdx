突然产生的好奇，在2500年前没有任何科学技术的情况下为什么德谟克利特会想到"物质不是无限可分的"？？他为什么能这样坚持并且说服别人接受没有任何证据的推测？居然还猜对了？

编辑，不是执着于原子这个词语，问题是"物质不是无限可分的"这个思想，"原子"只是他创造的一个概念，是这个思想的表象而已。我比较好奇他怎么会在2500年前坚持认为物质不是无限可分的
