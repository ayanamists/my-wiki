1.  每个月最多只有31天，如何确保一定会有赢的那一天？可能从月初输到月末哦
2.  不考虑1.的问题，玩到一直赢为止。那你这个每次×2的话，需要的资金随着天数是指数增长的（注意这个基本没法避免，你只能找个增长更快的，不能找增长更慢的，因为要保证
    a_n \>
    S\_\{n-1\}），那么这样一来玩到几轮后你发现下一轮需要的钱就会是一个天文数字了。
3.  1.、2.都不考虑，那"看上去"确实稳赚不赔。但我们没法不考虑1.2.，也就是说，这种稳赚不赔是建立在无限时间和无限金钱上的。
4.  实际上，这个就是某种贪心算法，找到局部最优解。如果能找到就是赚的，但是问题决定了它有可能找不到这个局部最优解。
</Reply>
