PDF
这种格式本身就是为了最终展示以及打印设计的，无论是编辑它、还是把它转换回某种用来编辑的格式，（比如
Word, html 等等）都是某种 Hack. 我们在 99% 的场景下要避免对 PDF 进行
"处理"。

所以，这个问题更像是
"为什么编译器大部分都是自由软件，而反编译器很多都是专有软件"。最简单的回答就是，反编译器不如编译器需求广，为了小众需求而开发的人，更倾向于获得金钱回报。
