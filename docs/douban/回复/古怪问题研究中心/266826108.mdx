我印象中，在八组还是瓜组，看过一个关于计划生育一孩时代，农村二女户的讨论。

简单来说，那就是，农村户口的家庭，第一胎是女孩的话，那这个农村家庭是可以合法生二胎。

二孩还是女孩的话，在孩子读书上是有一定的政策优惠的。

那个帖子的阵营大致上分为两派，一派认为，因地制宜，这个政策让更多的农村女孩活了下来。

一派认为，歧视，凭什么？女孩不是人，可以生二孩？

就我自己的个人观点而言，我觉得，从结果来看，这个政策让更多的女孩活了下来，尽管她们也会早早辍学打工，也会早早结婚生子，但是，至少，在她们有可能出生即死亡的背景下，她们活了下来。
