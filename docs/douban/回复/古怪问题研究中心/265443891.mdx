今天骑自行车路过一个高架桥下面，看到桥下面有一块铁杆之类的东西被路过的大车震的不停晃动，脑海中突然就联想到

如果这个时候发生意外，高架桥要塌了，而我是唯一一个发现端倪的人，或者有突然的预知感，周围所有路过的行人都毫无察觉，我会不会停下来不顾一切的让所有人退后远离，结果是只有我在危险的桥洞下被压死。

还有一次在新闻里看到矿洞坍塌事故，晚上做梦我就也成了被困一员，有一个只能一人爬行出去的特别长的小洞，特别危险，但是那是和外界取得联系的唯一途径，在梦里我就觉得那个人必须是我，然后我特别理所当然的去了，在洞里不知道爬了多久才爬出去，也因此解救其他人，而我却因为体力不支抢救无效死了。

梦醒了之后我就在想，我真的会有那样的勇气吗，真的有那样的信念感吗？这是我的英雄主义情结，还是只是因为我想死，但不想默默无闻，而是想死的伟大呢？
