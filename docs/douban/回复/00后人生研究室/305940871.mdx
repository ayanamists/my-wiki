现在学的进度大概是在做第二个项目，第一个项目好像太拉了就是一些crud，没什么亮点难点，写在简历上好像hr都不爱看，但我也不知道怎么改进😭😭

想问问看有没有前辈是学计算机但不从事本专业岗位的呢？想趁秋招前努力找个实习（不过很多岗位也不要不对口无经验的😭）就是想问问出路在哪

简历没啥也问题不大吧，一般 HR 看到 985 就 OK
了，不会卡你的。问题主要还是面试的时候要能和面试官聊起来，让他觉得你还不错。（不过最终还是看公司要不要找人～）

我确实觉得 Java 服务端不是很好的一个方向，不过我自己做的
PL（编程语言，比如写编译器/分析器）找不到啥工作，也没什么建议可以给你。如果说就业市场里比较多的工作，我个人还是最喜欢前端，但是现在也很多人唱衰......总之找工作不用太紧张，多找找总会有的。至于到底要不要转行，我认为唯一的判断条件就是你喜不喜欢写代码。如果你喜欢写代码，喜欢看到自己的程序运行，喜欢理解自己和别人的程序，那我还是劝你找一份写代码的工作；如果不喜欢就不要折磨自己了。
</Reply>
