和你用啥网没关系，和你用什么手段有关系。不过理论上来说，无论你怎么做代理，总是要和一台、几台、或者过一个或者几个域名（用
websocket 的话）眉来眼去的，网关是可以猜测这个人大概率是在搞事情的。

不过对于网关（学校也有网关）来说，只要你用的是比较新式的手段，他也是只能「猜测」你在干这个，是没法知道你具体上了哪些网站、做了什么事情的。但一般同学往往在
DNS 上不够谨慎，不知道要用 DOH 之类的手段去隐藏 DNS
查询，那学校就可以知道你上了什么网站了（但这个也只是「猜测」）
