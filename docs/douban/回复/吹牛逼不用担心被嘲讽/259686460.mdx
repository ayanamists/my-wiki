谢谢各位uu的夸奖😭来不及一一回复了，这个一年前的帖子能再被顶上来是我没有想到的，楼主现在高三，很幸运能得到楼里uu的夸奖

---------分------------割---------------线------------

lz高一，从小写字就很好看，临摹东西很像，而且没怎么练过，四年级的时候连笔锋是什么都不知道，趴在飘窗上抄了首春夜喜雨交上去拿了全国三等奖。有个擅长书法的学校老师劝我去学，一句话我记到今天："爱因斯坦说天才是百分之九十九的汗水加上百分之一的努力，而你是有那百分之一天赋的孩子。"

然后六年级时学了一年的软笔，当时软笔老师开始再三问我有没有学过国画，因为书画同源，他觉得我的字看起来不像是没摸过毛笔的人

![上上周第一次写瘦金体](https://img3.doubanio.com/view/group_topic/l/public/p537665623.webp)

初三平常的字就像成年人一样语文老师在全班面前直夸是可以出字帖的程度（当然这话有夸张成分）我写字也不想结构之类的，就是觉得这个笔画该写在这个位置，完全凭感觉。。所以有人问我怎么写的我真的回答不出来。。。
