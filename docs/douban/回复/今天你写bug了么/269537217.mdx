高考报志愿时听信工资高，也不清楚自己到底喜不喜欢就一股脑全报的计算机类专业，最后被北京一所末流211录取到了软件工程专业。
不过当时确实不知道自己喜欢什么，但是这四年大学我真的没从该专业收获过一丝成就感和快乐，只获得了源源不断的自我否定。
期末刷往年真题抱佛脚，课程设计找大佬组队抱大腿，疫情期间网课在家划划水。虽然没挂过科，但是也从没有主动学习的热情，大学四年我什么都没学会，现在就是five一个。
我自知绩点保研无望，也尝试了科研竞赛，期间拿过几次国家级竞赛的奖项，凭此我可以争取学校的创新保研名额，但是需要先通过答辩面试。唉，当时的情况是今天出通知，明天就答辩，后天出结果。我没准备好，老师的问题没答好。结果是16个人进了8个，我落榜了。
大三下，我意识到好像只有考研这一条路了。奈何我太没有自知之明了，对目标院校上岸难度没有足够的认知。去年它从英二数二改成了英一数一，而且专业课还是408，四门巨深巨难的专业课（对我来说），其中最难且分数占比最大的计算机组成原理还不在我们专业的教学计划之内，完全靠自学，而我还学不明白。
唉，是我对自己的认知不足选高了目标，并且我备考期间的专注度不够，虽然我不打游戏不追剧不看小说也不去玩，但我就是没有完全潜下心来学习。结果就是复习不完了，十一月份冲刺期间我几度崩溃，自觉我肯定考不上，当时状态差到了极点。
毫无疑问，初试分数根本连目标院校的边都沾不到，虽然都过了国家线，但是也不可能调剂到稍微好点学校的相关专业，我也不想二战。至此，我的考研彻底结束了。
大四下延迟开学，bj疫情又没能返校。就这样在家待了半年，除去搞毕设的时间我完全就是混沌度过的。我知道要春招，但我觉得自己什么都不会，始终没有勇气去投递简历。肯定会骂我为什么不好好准备一番呢？唉，我当时独自在家过于舒适了，就一个混子的心态，过得跟咸鱼一样的生活。我也很想打醒当时的自己，但过去既定事实，后悔已无意义。
勉强搞完了毕设，除了双证我什么都没有，甚至连一张和学校的合影都没有。我不知道我该怎么办，也不知道我的前路在何方。
我现在有下面一些想法和与之伴随的担忧：
1、在家努力自学一门技术，努力刷题，这个过程可能要好几个月之久。
我应该不会去培训班，因为lz农村女，父母农民工，家里这两年刚全款买了房子也搞了装修，还有一个马上要读民办二本或复读的弟弟，家里没钱给我去上培训班。lz自己的积蓄只有几千块，也不够。
担忧：自学了还是找不到工作😭
2、在家复习，准备二战。离23考研还有6个月左右，如果二战我会找一个两年学制且考试科目不多的目标院校。
为什么我不提前准备二战呢？因为我之前不打算二战，我觉得我们这行工作经验更重要，而且99年10月份的我有些年龄焦虑。错过了学校二学位招生就没法去学校备考了。
担忧：在家没有学习的氛围，但想不出还有其它适合的地方。不确定性很大，如果二战也失败了该怎么办呀😭如果考上了的话就多一份硕士学历，多一次校招的机会，多两年时间准备。
3、跨考其他专业，从事其他专业的工作，难度好像更大。唉，我好像没有找到第三条出路。
我真的不知道该怎么办了，如果友友们有什么建议和批评都请砸向我，它们都是指引我的明灯。有任何想法也都可以在评论区分享。
谢谢大家看到了这里，我写得确实有点多了。
