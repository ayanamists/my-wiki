就是，sl上认识一个小姐姐，某211哲学系今年刚毕业，很漂亮，很哲学，喜欢黑格尔，暗恋她老师，有故事，m倾向。我恋爱脑，疯狂喜欢她，第一次见面一起聊哲学，请她喝咖啡，逛街，给她买了一条五百多的裙子，然后重点来了：

她说可以给钱，然后那啥，看我能给到多少，她说这是近期才有的想法，不想当男人的工具，需要钱来找到价值。我拒绝了，我不想变成交易，虽然我也不差钱。后面又谈了好久，谈专业也谈别的，一点半见面七八点才回去。

然后，接下来的一周我疯狂想她，上头了，我还写了一篇小说，我给她发了好几条消息，一周了杳无音信，是不是她已经出去挣钱了。我该怎么办。

---------------------------------------------------------------

更新，周日突然又联系我见了个面，我俩都没啥想去的地方，就带她做了个美甲，逛了个超市，牵手，拥抱，她也揽着我，她的手很温热。然后就各回各家了。聊了一下专业，她嫌弃我只买书不看书，说我书搜集够多了，都快赶上哲学系研究生了，但是不深入到内部看，犯了一个哲学初学者的大忌。我送了她《哲学论稿》，她说看完还我，胡、海的书基本看完了，因为里面有很多重复的东西，现象学推荐倪梁康的《概念通释》，《现象学的观念》要看新版。她说接下来要看黑格尔了。我解释我工作太忙，考研也还是文学老本行，我也就是业余看看西哲，她说先从柏拉图开始看吧。

---------------------------------------------------------------

感谢大家的建议，我可能是太喜欢哲学了吧，我曾经因为读不懂哲学而抑郁，现在文学和历史的学位都有了，哲学是个永远的白月光。在我们这个三线不到的城市，能遇到这样的姑娘，自动就加了滤镜，当然也因为她长得好看，还有就是我喜欢这种有过创伤和故事的女生，同情心极度泛滥不考虑后果。大家的建议我会好好考虑，出于安全考虑，对自己和亲人负责，我也不会轻易和别人发生什么的。目前就是顺其自然吧，可能突然有一天，我就不会那么迷恋她了吧，再次感谢大家。

---------------------------------------------------------------

这周我也想了很多，我邀请她出来一起看书，她没有出来，也没有回复我的消息。她的戒指还在我这里，我小心翼翼搁在漂亮的漆盒里，或许以后可以见到她，或许她不会再叫我出来了。我觉得我这个故事可能是烂尾了，烂尾可能是最好的结果，我不愿意相信她是姬，起码在我这里她不是，我是馋她的身子，但是我更欣赏她把哲学融进生活的勇气。她的从小养成的性格就很抑郁，因为抑郁她热爱思辨，又因为思辨让她抑郁，她的生活和哲学是一体的，她说离开哲学她就会死，那时我总会想起把自己生命交给人类未知边缘的福柯。她说她最想成为那架失事飞机上的乘客，不用负责任地离开。她说过段时间她要走，她不知道去哪里，但是要离开这个城市。她说，词语破碎处，无物可存在。我在她身上，我看到了所有诗意，那是我年轻时在小说里见到过的，我无法不动容，我看着自己这副皮囊，我厌恨自己懒惰而平庸，我如同蝼蚁一样，攀爬在黄尘黑土里，拿什么拯救她，她需要我卑微的救赎吗？

星空啊星空，那是康德的道德律，那颗莹莹血泪的流光，照着我流泪的眼，她破碎的心。

---------------------------------------------------------------

谢谢大家一直以来的关心，今天她联系我了。简单聊了一下，细节我就不复述了，抱歉大家，我和她的故事可能就是这样了，后续应该也没什么后续了，我把在这里写的最后一个片段发给她看，她说挺好的。附一张聊天截图，好好告个别吧。

---------------------------------------------------------------

故事结束了，我们删了微信。
