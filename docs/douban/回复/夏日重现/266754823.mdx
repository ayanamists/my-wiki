人民日报1965年8月9日：

这次大会是在美帝国主义在广岛和长崎投下两颗原子弹二十周年期间举行的。美国投掷的原子弹祸害了日本人民，但是吓不倒日本和世界人民，反而促使各国人民更清楚地认识到美帝国主义是日本和世界人民最凶恶的共同敌人，并且更加强有力地团结起来打击美帝国主义的核讹诈政策。美帝国主义和为它效劳的苏联领导一手制造的莫斯科三国条约的骗局也遭到了严正的谴责。许多代表在发言中深刻地论述了从本质上区别核武器掌握在谁手里的这个重要原则问题。中国核试验成功得到代表们的热烈祝贺。他们赞扬中国的核武器是人民的原子弹，是用来保卫世界和平的，对粉碎美帝国主义的核垄断、制止美帝国主义发动核战争有着十分重要的意义。中国对禁止核武器的严正主张，普遍地受到代表们由衷一致的热烈称赞。大会在决议中明确宣告："在帝国主义者反对禁止核武器、进行核战争挑衅的当前情况下，不应把帝国主义用于侵略目的的核武器同社会主义国家和新兴独立国家为了防御而拥有核武器混为一谈"。这一宣告，对形形色色的修正主义分子不啻是致命性的当头棒喝。
