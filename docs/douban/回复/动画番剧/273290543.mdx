《空之境界》真的很好看很好看，但组内很少有人介绍为什么它如此吸引人，于是开个帖探讨一下。

警告⚠️⚠️⚠️： 本番含有重口xue腥画面，请酌情观看，或选择b站和谐版本。

值得安利的点：女强，怪谈世界观，优秀的镜头语言，梶浦由记巅峰期bgm

《空之境界》同属于型月世界观内，比起《Fate》系列的宏大叙事，《空之境界》更像是都市怪谈，但在各类猎奇案件之下，仍然是奈绪蘑菇不变的主题------boy
meets girl。

一个是黑道大小姐出身，情感因为丧失记忆而淡漠（其实是失去了一个人格），得到了看见万物之死的直死之魔眼，感官上与世界产生隔阂的女主两仪式。

![image](https://img3.doubanio.com/view/group_topic/l/public/p566563047.webp)

一个是平凡辍学大学生，表面平平无奇，实际上不自觉靠本身的魅力吸引了三个boss的亚撒西男主黑桐干也。

![image](https://img3.doubanio.com/view/group_topic/l/public/p566563042.webp)

啊，简称暴力黑道大小姐和她的小娇夫。（顺带一提男主最后入赘了女主家，真的日漫里世间罕见）

但《空境》绝不是一个落于俗套的玛丽苏的恋爱故事。

从表面上来看，《空境》讲述的是男女主合伙调查在城市内发生的各类sha人魔事件，实际上对各类社会事件，无论是经济，校园还是对人类本身的存在都有哲学的探讨。

但在我看来，男女主不管是人设还是感情进展都是日漫中很少有的。

女主两仪式，飒爽的短发和服美女。因为体内男性人格的死去所以在刚出场时经常不自觉模仿男性口吻，表面看上去冷冰冰，实际上和她熟悉后有那么点傲娇。

![猫咪一样！](https://img3.doubanio.com/view/group_topic/l/public/p566563043.webp)

战力一绝，贡献了绝佳打斗场面，以及台词"只要是活的东西，是神我都杀给你看。"

![帅气的美人](https://img1.doubanio.com/view/group_topic/l/public/p566563048.webp)

虽然男主黑桐干也的人设是"普通""温柔"，但他真的在我心中重新定义了"温柔"------给三年昏迷不醒的女主每星期都送上花束；把路上遇见的看上去身体很难受的女孩子提供住所，自己选择出去住；不相信女主是sha人魔（实际上连女主自己都不确定自己有没有sha人），彻夜蹲守在女主家门口。

![罪孽深重的男人.jpg](https://img3.doubanio.com/view/group_topic/l/public/p566563037.webp)

虽然很多人戏称《空境》的剧情就是两仪式在各个案件里干掉情敌，但是我真的能够理解那些女配（甚至男配）对男主有朦胧的好感，因为对于与疼痛一生相伴的他们来说，男主真的是令人向往并追逐的温暖的光。

《空境》的镜头语言也很迷人。

![追逐蜻蜓，但最终力竭坠落的蝴蝶](https://img1.doubanio.com/view/group_topic/l/public/p566563039.webp)

![少女的口脂](https://img1.doubanio.com/view/group_topic/l/public/p566563040.webp)

![名场面](https://img9.doubanio.com/view/group_topic/l/public/p566563044.webp)

《空境》的观看门槛有那么一点高，包括且不限于：

1.各篇章时间线的错乱，即各个篇章发布的顺序与故事发展的时间线是不同的。b站上有up主整理过按照故事的时间线来看的观看顺序。

但要是问我哪个顺序看比较有体验感的话，我还是推荐对这个故事有些基本了解再按照发布顺序看比较精彩。

2.菌言菌语。简称不说人话。因为世界观中涉及到了型月特有的"魔术"概念，虽然作者已经很尽心尽力地用各类通俗易懂的比喻来解释各类事件的发生了，但一旦缓过来仍然让人思考"我是谁，我在哪里，这又是为什么"。

所以它绝对不是一个不带脑子就能看的番（痛苦）

但同时这样叙述又很有魅力。就像一本读不厌的诗歌。忍不住反复地观看，反复揣摩人物的心境，不断地被吸引。

到最后真的算我看得最多的一部番了。
