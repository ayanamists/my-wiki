------------------还是不会打分界线------------------

和上小学的女生聊天，发现她和班里的女同学很多都在看斗罗大陆，还觉得男性凝视是正常的审美。她就是那种比较喜欢看言情类的温和的影视文学作品的那种，我觉得会喜欢这种类型的动漫有点奇怪。

本身就是厕所读物，好比在贴吧当女权玩手游找对象，不是没有大浪淘金，而是没必要沙子里找金子

------------假装是分界线------------------

我小学时候也会借男同学的斗罗大陆之类的小说和漫画，不过纯纯为了漫画画风好看+看现实中的衰男意淫很好笑，没错，当时小小年纪的我就觉得只有普信下头男才会写出这样的剧情或者真心喜欢这样的剧情。

当时还有很多同样类型的小说，就是现在男频小说啊，我觉得没什么特别的地方，不过唐三确实很能更新不会弃坑罢了。

还从同学那里借着看过唐三的其他小说，就是单纯的黄书（情节不是很多）+意淫。

不太明白为什么会有人批判它，我对它的要求，嗯，就是没有要求。

但是一些年纪小的女生还很喜欢看，并且没搞清楚它的定位，我不是很理解。

有些类似的网文作家很赚钱很成功，我觉得合情合理，因为也确实给人提供了乐趣嘛，而且好像唐三在采访中也表示自己不会再在写作上提高，写作就是要迎合市场偏好。他的市场定位很清晰，而且也假装任何高端局，问题是一些读者/观众好像对这些作品没有认知？
