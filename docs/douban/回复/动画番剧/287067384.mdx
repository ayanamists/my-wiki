作者想表达的事情不是一句话两句话能说清楚的，简单说说我比较有感触两个点：

1.  伪物语结尾的问题：「这是贝木君以前经常玩的思考游戏------假设有一个真物，以及一个与真物相似到无法分辨的伪物，哪个更有价值？对于这个问题，咱的答案是当然真物比较有价值。忍野君则说两者拥有同等价值。但是贝木君的答案则是，伪物才具有压倒性的价值。只要拥有成为真物的意志，伪物就比真物还要来得真实。」贝木的这句话真的很有一种中二的帅气感......

2.  恋物语，这是我觉得物语系列最好看的一章。贝木和战场原黑仪那种若即若离的感情，以及贝木最后和抚子对决的时候说的一堆话都很有意思。

3.  猫白结束以后班长黑白相间的头发真的很好看
