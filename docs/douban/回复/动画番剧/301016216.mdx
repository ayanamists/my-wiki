0079→Z→ZZ→CCA→闪哈，除了CCA因为富野报社导致跟前后不连贯，期待有朝一日贝子动画化，有梦想总是好的(。但CCA拍得真的好，绝对秒杀一众好莱坞商业片。

虽说观念很昭和(米莱这样的神舵手居然结婚后当家庭主妇了)，但也有塞拉这样的人生赢家......炼铜问题感觉是年龄设定都偏小了，拉拉哈曼葵司外表和心智怎么看都不像未成年，同理老炼铜术士夏亚在0079里怎么看都30+了(感觉更bt了😅)

没有战斗力迭代爆表问题，主角有光环但仅限开局，各路势力都很均衡。除了有点昭和，世界观和故事我感觉都挺不错，在日漫里还没看到更好的。

PS
CCA我觉得好在制作和节奏，节奏真的很catchy，对MS没兴趣的朋友都看得津津有味。但台词和一些观念以及炼铜的确问题很大(。夏亚的确一辈子都在装莫名其妙的b但遗言实在是离天下之大谱，还有自以为是的zng装深沉说什么此话一出夏亚和阿姆罗终于达成了最终的理解，tm阿姆罗的反应明明是(跟观众们一样)被雷得外焦里嫩，死不瞑目......
