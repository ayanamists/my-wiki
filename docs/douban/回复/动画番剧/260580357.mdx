最近重温了美少女战士，发现剧情里面跟现在完全不脱节，而且很前卫，不管是时尚，音乐，还是性向等。

而且女性想向的恋爱番对感情叙事都比较细腻，相反与男性向更偏向于肉体。

但是近几年来的番女性向好像越来越少，这种肉体，卖肉反而成为二次元现在的一种趋势(就好比Kpop曾经流行一段时间的性感风？)。

我已经入坑二次元好多年了，但近几年看的番越来越少，我以前看的女性向番现在找不到类似的了🤧🤧以前的女性向真的是经典中的经典啊!

爱丽丝学园，梦色糕点师，真珠美人鱼，库洛魔法师等等都承包我的童年，现在真的找不到了🤧🤧🤧🤧

所以如题一样，是什么原因让女性向的番越来越少呢🤧

------------------------------------------------------

各位讨论的很激烈，我赞同一些观点比如:

1日本的女性社会

2日本近几年的经济

3高质量地女性向恋爱番已经减少

还有很多人提到真正地女性向一定要男主是帅又强，而在男性向恋爱番普遍是女强男弱。

那我解释一下经常说男性向的恋爱番的女强男弱。
大部分的"女强男弱"都是提现在男主特别废，特别丑，女主特别美，身材好，高人气仅此而已。
我也不喜欢有些动漫把女性角色画的也特别废一无是处。我希望在恋爱关系中两人是持平的。并不是一方特别弱一方特别强势，相反男主特别强势就给我三次元大男人主义的既视感。
关于男主只能帅这点我保持中立，因为我也不是不接受男主长得平凡。男主长得普通，能尊重女主，除了颜值上还有其他的闪光点我非常能接受。相反的长得帅却不尊重女主一脸爹味我觉得没好到哪里去。
我其实要讨论的是，像以前一样高质量的女性向恋爱番已经减少了，可能我表达的不是很清楚
