自从 Bilibili
实行先审后播之后，如何观看最新的日本动画就成了一个亟待解决的问题。诸如
Age.net 之类的小网站为了节省带宽资源，无一例外地使用了码率非常低的片源。
所以，时隔多年以后，Bt
下载又成为了看动画的第一选择。但我们不止想要下载，更想要一种自动化的播放体验：



要达成这种体验，自然是需要一整套软件系统的。首先，我们需要一个「媒体服务器」，它可以把硬盘上的番剧变为可通过
web 或客户端点播的资源。

[Jellyfin](https://www.douban.com/link2/?url=https%3A%2F%2Fjellyfin.org%2F&link2key=8cb9c9d33b) 是我的选择。通过
[jellyfin-plugin-bangumi](https://www.douban.com/link2/?url=https%3A%2F%2Fgithub.com%2Fkookxiang%2Fjellyfin-plugin-bangumi&link2key=8cb9c9d33b)，jellyfin
可以获取番剧的中文元数据，进而给出非常良好的番剧归档：

![image](https://img3.doubanio.com/view/group_topic/l/public/p594410593.webp)

![在这个界面就可以播放了](https://img1.doubanio.com/view/group_topic/l/public/p594410748.webp)

那么，问题就剩下了「如何自动下载番剧」？对于这个问题，我的选择是
qBitorrent 的 Rss 订阅功能。如果想要全自动体验，那么可以使用
[Auto_Bangumi](https://www.douban.com/link2/?url=https%3A%2F%2Fgithub.com%2FEstrellaXD%2FAuto_Bangumi&link2key=8cb9c9d33b)。如果能接受半自动（手动导入订阅列表、可能需要手工重命名剧集），那么可以直接订阅
mikan.com 或 dmhy.org 的 rss 列表
